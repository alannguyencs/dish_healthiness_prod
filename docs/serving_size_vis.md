# Serving Size Reference

## Standard Measurements (FDA/USDA)

| Unit | Metric Equivalent |
|------|-------------------|
| 1 cup | 240 mL |
| 1/2 cup | 120 mL |
| 3/4 cup | 180 mL |
| 1 tablespoon (tbsp) | 15 mL |
| 1 teaspoon (tsp) | 5 mL |
| 1 fluid ounce | 30 mL |
| 1 ounce (weight) | 28 g |
